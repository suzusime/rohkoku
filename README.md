# rohkoku
ラズベリーパイで動かすために作った時計です。解像度は 480x320 px です。

[sdl2quick](https://github.com/ohai/sdl2quick)を使っています。

## セットアップ
Ruby（2.6.5で動作確認しています）と、Rubyのgemの

- [ruby-sdl2](https://github.com/ohai/ruby-sdl2)
- [wareki](https://github.com/sugi/wareki)

のインストールが必要です。

起動は

```sh
$ ruby get_news.rb
$ ruby tenki.rb
$ sh ./start.sh
```

です。
